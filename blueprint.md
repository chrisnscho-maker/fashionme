
# Project Blueprint

## Overview

This project is a web application that allows users to input their physical information, including photos, weight, and height. It consists of a main page for basic information and an advanced page for more detailed photo uploads.

## Project Outline

### Style and Design
- **Layout:** Simple and clean, with a focus on usability.
- **Styling:** Basic CSS for a clean and modern look.
- **Colors:** A simple color palette will be used.
- **Fonts:** Standard web fonts will be used.

### Features
- **Main Page (`index.html`):**
  - Profile picture upload.
  - Weight and height input fields.
  - Link to the advanced details page.
- **Details Page (`details.html`):**
  - Bust shot photo upload.
  - Lower body photo upload.
  - Full body photo upload.
  - Link back to the main page.
